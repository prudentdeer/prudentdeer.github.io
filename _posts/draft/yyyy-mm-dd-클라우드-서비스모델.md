PaaS (Platform as a Service)
  - 개발에 필요한 운영체제, 개발환경, 개발툴, 프레임워크 등을 오토스케일링 제공하는 서비스
  - 오토스케일링 
  - 애자일 기반 개발자에게 유용
  - 구글 앱 엔진, 윈도우 Azure
SaaS (Service as a Service)
  - 소프트웨어 자체를 비설치형, 임대방식으로 멀티테넌트 제공하는 클라우드 서비스
  - 멀티테넌트  
  - 설치하는 대신 웹 임대 형태   
  - 구글 앱스, 드롭박스
SecaaS (Security as a Service)
  - 보안에 필요한 모든 요소들을 클라우드 형태로 제공하는 서비스
  - 보안조직 운영부담 절감
  - 최신 보안데이터 적용
