          기술요소 제레오스
  JSON, XML - 구조화, 문자열화된 문서 데이터형식 또는 경량데이터 형식
  REST - URI 를 통해 자원명시 및 자원에 대한 POST, GET, PUT 등 명령을 주고받는 방식
  OAuth 2.0 - 외부 서비스에서 Access Token 기반 제한된 자원접근, 인증을 위한 표준
  SSL/TLS - 통신하는 데이터를 안전하게 보호하기 위해 인증 및 비밀키 암호화 사용 프로토콜
