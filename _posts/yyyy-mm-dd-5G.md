          <table class="post-table">

          특징 전체주고 지연에면
  최고 전송속도 : 20Gbps
  이용자 체감 전송속도 : 100~1,000Mbps
  주파수 효율성 : 4대비 3배
  고속 이동성 : 500km/h
  전송지연 : 1ms
  최다 기기 연결 수 : 106 개/km2
  에너지 효율성 : 4G대비 100배
  면적당 데이터 처리용량 : 10Mbps/m2
코어망 핵심기술 NCNS
  NFV / 기능가상화 - 물리적 하드웨어 네트워크 기능 정의하는 SW를 분리
  C-RAN / 중앙집중처리 - 데이터를 각 기지국 공유해 중앙집중 처리방식 아키텍쳐
  Network Slicing / 논리적망분리 - 서비스별 특화된 네트워크망 제공
  SDN / Control, Data 분리 - 네트워크 장비에서 컨트롤 플레인과 데이터 플레인을 분리
무선전송기술 MMM FIN
  Massive-MIMO / 다중입출력 - 수십 개 이상의 안테나를 이용하는 다중 입력 다중 출력 기술
  MHN / 고속이동환경 - 광대역 주파수를 활용, 고속 이동 환경에서도 기가급 속도 제공
  MEC / 클라우드적용 - 엣지 단에 응용서버 설치, 클라우드 기술을 적용한 End-to-End 기술
  FBMC / 간섭최소화 - Filter Bank를 설치하고, Side Lobe를 제거하여 반송파간 간섭을 줄인 기술
  IBFD / 전이중통신 - 동일 주파수 대역 데이터 송수신을 동시에 하여 스펙트럼 효율 향상시킨 기술
  NOMA / 비직교다중접속 - 동일한 주파수/시간/공간 영역을 공유하는 비직교 다중 접속 방식"
